# Čerč–Tjuringova teza i granice izračunljivosti

## Autor
Mihajlo Končar

## Opis projekta
Ovaj repozitorijum sadrži studentski projekat koji obuhvata web prezentaciju
i akademski rad na temu Čerč–Tjuringove teze i granica izračunljivosti.
Projekat je realizovan u okviru fakultetskih obaveza i koristi HTML, CSS i LaTeX.

Web deo projekta sastoji se od više međusobno povezanih stranica koje prikazuju
osnovne informacije o temi, dok je teorijski deo predstavljen kroz PDF dokument
generisan pomoću LaTeX-a.

## Struktura projekta

prezime_ime_brindeksa/
- index.html
- tema.html
- kontakt.html
- style.css
- main.tex
- rad.pdf
- README.md
- slike/

## Pokretanje projekta
Web prezentacija se pokreće otvaranjem fajla `index.html` u web pregledaču.
Navigacija između stranica dostupna je putem menija na vrhu stranice.

PDF dokument se može otvoriti direktno iz fajla `rad.pdf` ili putem linka
na web stranici.

## Napomena
Svi resursi u projektu su lokalni. Kontakt forma je demonstracionog karaktera
i ne vrši stvarno slanje poruka.
